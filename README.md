### 📝 About the Project
This repository showcases a modern front-end landing page for **AirPods Max** headphones. The main goal of the project is to demonstrate clean layout design and native interactions without relying on heavy frameworks or external libraries.

### 🚀 Key Features
* **Vanilla Stack:** Built entirely with pure HTML5, CSS3, and basic JavaScript.
* **Responsive Design:** Optimized for a seamless experience across desktops, tablets, and mobile screens.
* **Smooth UX:** Includes fluid CSS transitions, interactive elements, and micro-animations to enhance user engagement.

### 📝 О проекте
В этом репозитории представлена современная верстка страницы для наушников **AirPods Max**. Главная цель проекта — продемонстрировать создание визуально привлекательного интерфейса и нативной интерактивности без использования тяжелых фреймворков и сторонних библиотек.

### 🚀 Ключевые особенности
* **Чистый стек:** Проект реализован исключительно на HTML5, CSS3 и базовом JavaScript.
* **Адаптивность:** Интерфейс полностью оптимизирован для корректного отображения на ПК, планшетах и мобильных экранах.
* **Плавный UX:** Реализованы плавные CSS-переходы, интерактивные элементы и микроанимации для улучшения пользовательского опыта.
